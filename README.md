# Apex Ride

Premium car rental platform built with React, TypeScript, and modern web technologies.

## Features

- Luxury car browsing with 3D previews
- Real-time booking system
- Dashboard for hosts and users
- Responsive design with animations

## Quick Start

```bash
cd client
npm install
npm run dev
```

## Tech Stack

React 19, TypeScript, Vite, TailwindCSS, Framer Motion, Zustand, Three.js
