## 🚀 **What is This?**

Apex Ride is a cutting-edge luxury car rental platform designed for discerning drivers across Bangladesh. Experience the pinnacle of automotive engineering through our meticulously crafted digital interface that combines stunning visuals with seamless functionality.

### 🎯 **Core Features**

- **🚗 Luxury Fleet**: Hypercars, sports coupés, and electric tourers
- **🎨 3D Visualization**: Interactive car previews powered by Three.js
- **⚡ Real-time Booking**: Instant reservations with live availability
- **📊 Smart Dashboard**: Analytics and management for hosts/users
- **📱 Responsive Design**: Perfect experience on all devices
- **✨ Smooth Animations**: Framer Motion powered micro-interactions

## 📦 **Installation**

```bash
# Clone the beast
git clone https://github.com/salahuddingfx/apex-ride.git

# Enter the den
cd client

# Install dependencies
npm install

# Fire it up
npm run dev
```
